# Creator Rewards

### Build games people love. Hit milestones. Earn tokens.

---

## The Creator Flywheel

```
Build great games  →  Attract players  →  Hit milestones  →  Earn tokens  →  Build more games
```

EAISports rewards creators for building games that attract real player traffic. As unique visitor counts grow, games unlock milestone rewards automatically.

---

## Reward Structure

### Per-Game Milestones

Each published game can unlock the following rewards:

| Unique Visitors | Reward |
| --- | --- |
| 100 | 500 `$EAISPORTS` |
| 500 | 3,000 `$EAISPORTS` |
| 1,000 | 8,000 `$EAISPORTS` |
| 5,000 | 50,000 `$EAISPORTS` |

These are **milestone rewards**. Each threshold is earned **once** when the game reaches it. Milestones are checked hourly by the platform.

### Multi-Game Stacking

There's no limit to how many games you can publish. Each game has its own independent milestone track, so creators with multiple successful games compound rewards over time.

**Example:** 3 games each hitting 1,000 visitors = 3 × (500 + 3,000 + 8,000) = **34,500 `$EAISPORTS`**

## Maximizing Rewards

### Build for Retention

Games that keep players coming back create more opportunity for traffic growth. Consider:

- Replayable mechanics (high scores, randomized content)
- Clear skill progression and short session loops
- Social hooks that make players share or challenge friends

### Use Leaderboards and Referrals

Leaderboards create replay pressure. Referrals create acquisition loops. Together, they increase the odds that a game reaches higher visitor thresholds.

### Publish Often

Each game is a new opportunity to earn milestone rewards. Use skills to accelerate development — build and publish a new game in under 30 minutes.

### Go Viral

Share your games on social media. Every game page has built-in sharing for X (Twitter) and direct link copying. The Windows XP aesthetic tends to get attention.
