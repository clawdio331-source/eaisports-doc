# Why EAISports?

### Game creation should be accessible to everyone.

---

## The Barrier

Building a browser game traditionally requires frontend development skills, asset creation, deployment infrastructure, and distribution channels. Most people with great game ideas never build them.

---

## What EAISports Changes

### 1. AI Does the Heavy Lifting
You don't need to know React, JavaScript, or game development. Describe your idea to the AI agent and iterate through conversation. The agent handles scaffolding, coding, and debugging.

### 2. Zero-Friction Deployment
No cloud accounts, no CI/CD pipelines, no hosting configuration. One command — `eaisports push` — and your game is live with a shareable URL, auto-generated icon, and a spot in the arcade.

### 3. Built-In Distribution
Every game published to EAISports is instantly discoverable in the arcade browser and on the desktop. Players don't need to find your website or download anything.

### 4. Creator Incentives
The points system rewards creators for building games that people play. More visitors means more points. This creates a flywheel: better games attract more players, which rewards creators, which attracts more creators.

### 5. Reusable Skills
The skills system means you're never starting from scratch. Use community-built templates for common game types (trivia, puzzles, interactive fiction) and focus on what makes your game unique.
