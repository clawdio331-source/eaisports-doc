# What is EAISports?

### The AI-powered game creation platform.

EAISports is a complete platform for building, deploying, and playing browser-based games — powered by AI from start to finish.

---

## Three Components

| Component | What It Does |
| --- | --- |
| **CLI** | An AI agent that builds games through conversation. You describe what you want, it writes the code. |
| **API** | Backend service handling game deployment, creator profiles, analytics, and the skills registry. |
| **Web** | A retro Windows XP-themed desktop where players browse and play published games. |

---

## Who It's For

**Creators** — Anyone who wants to build a browser game, regardless of coding experience. The AI agent handles the implementation; you guide the creative direction.

**Players** — Discover and play community-built games directly in the browser. No downloads, no installs.

---

## Key Features

### AI-Powered Build Agent
Describe your game idea in plain language. The agent uses Claude Opus 4.6 (via OpenRouter) with a full tool suite — file operations, terminal commands, browser preview — to build a working React + Vite game.

### One-Command Deployment
Run `eaisports push` and your game goes live at eaisports.ai within seconds. The platform handles bundling, cloud storage, and icon generation automatically.

### Skills System
Reusable game templates (trivia, interactive fiction, puzzles, UI components) that seed the AI build process. Use a skill to get a head start, or extract patterns from your finished game into a new skill.

### Creator Economy
Earn points for every visitor milestone your games hit. Track your stats on the creator dashboard. Connect a Solana wallet to accumulate rewards.

### Windows XP Desktop
The web platform presents games as desktop icons on a nostalgic Windows XP interface — complete with a start menu, taskbar, draggable windows, and user profiles with avatar selection.
