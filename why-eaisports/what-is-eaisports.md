# What is EAISports?

### A game development studio powered by the AI you already pay for.

EAISports turns any AI subscription — ChatGPT, OpenRouter, or others — into a complete game creation and distribution platform. Describe a game, watch AI build it, deploy it in one command, and earn rewards as players show up.

---

## The Viral Loop

```
More creators → More games → More players → More rewards → More creators
```

EAISports is designed as a self-reinforcing ecosystem:

- **Creators** build games using AI and earn `$EAISPORTS` tokens when players visit
- **Players** earn points for playing, compete on leaderboards, and share with friends
- **Referrers** earn bonus points when their network plays
- **Everyone** benefits as the platform grows

---

## Three Components

| Component | What It Does |
| --- | --- |
| **CLI** | An AI agent that builds games through conversation. You describe what you want, it writes the code. Works with ChatGPT, OpenRouter, and 5 other providers. |
| **API** | Backend service handling game deployment, player tracking, leaderboards, referrals, and the skills registry. |
| **Web** | A retro Windows XP-themed desktop where players browse and play published games. |

---

## Who It's For

**Creators** — Anyone who wants to build a browser game, regardless of coding experience. Already have a ChatGPT subscription? That's all you need to start. The AI agent handles the implementation; you guide the creative direction.

**Players** — Discover and play community-built games directly in the browser. No downloads, no installs. Compete on leaderboards and earn points.

---

## Key Features

### Use Your Existing AI Subscription
Already paying for ChatGPT Plus or Pro? Connect it and start building games immediately. No new accounts or API keys needed — unless you want access to other models through OpenRouter or other providers.

### One-Command Deployment
Run `eaisports push` and your game goes live at eaisports.ai within seconds. The platform handles bundling, cloud storage, and icon generation automatically.

### Skills System
Reusable game templates (trivia, interactive fiction, puzzles, arcade, mobile-optimized, UI components) that seed the AI build process. Use a skill to get a head start, or extract patterns from your finished game into a new skill.

### Creator Economy
Earn `$EAISPORTS` tokens for every visitor milestone your games hit. Track your stats on the creator dashboard. Connect a Solana wallet to accumulate rewards.

### Leaderboards & Competition
Every game has daily, weekly, and all-time leaderboards with dynamic prize pools. Top players share rewards, creating a reason to keep playing.

### Windows XP Desktop
The web platform presents games as desktop icons on a nostalgic Windows XP interface — complete with a start menu, taskbar, draggable windows, and user profiles with avatar selection.
