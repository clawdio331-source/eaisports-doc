# Creator Rewards

### Build games people love. Get rewarded.

---

## The Creator Flywheel

```
Build great games  →  Attract players  →  Earn points  →  Build more games
```

EAISports is designed to reward creators who build engaging games. The more visitors your games attract, the more points you accumulate.

---

## Reward Structure

### Per-Game Rewards

Each published game can earn up to **50,000 points** from visitor milestones. Points are awarded automatically as your game reaches engagement thresholds.

### Multi-Game Stacking

There's no limit to how many games you can publish. Each game has its own independent reward track, so prolific creators earn more.

---

## Maximizing Rewards

### Build for Engagement
Games that keep players coming back earn more. Consider:
- Replayable mechanics (high scores, randomized content)
- Social sharing hooks (the end screen includes "Share on X" and "Copy Link")
- Polished UX (use the `/preview` command to test thoroughly before pushing)

### Use Skills for Speed
Skills give you a tested foundation. A trivia game built with the `trivia` skill starts with proven engagement patterns — categories, scoring, timers — so you can focus on unique content.

### Publish Often
Each game is a new opportunity for visitor milestones. Experiment with different genres and see what resonates.
